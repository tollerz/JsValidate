# JsValidate
<h3>jQuery Form Validation Plugin - Easy form validation</h3>

A customisable jQuery form validation plugin.<br>

<h4><u>Usage</u></h4>
<pre>
    $('form').validate({
        rules: {
            inputElementId: {
                minLength: 10
            },
            selectElementId: {
                required: 'Custom required message'
            }
        }
    });
</pre>
